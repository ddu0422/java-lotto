# Lotto Game

### 기능 요구 사항

- 로또 게임 기능을 구현
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급
- 로또 1장의 가격은 1000원
- 로또 당첨 금액은 고정되어 있는 것
- 수익률을 계산해 출력

### 기능 구현 목록

1. 구입금액 입력
오류) 숫자가 아닐 경우, 범위를 벗어날 경우

2. x개 구매 출력
예외) 구입금액 / 1000 이 0이하 일 경우 다시 입력

3. 로또번호 x개 생성
  - 로또 객체배열 생성
  - 로또 객체 생성
  - 랜덤 숫자 생성
  - 로또 번호 생성
  - List에 중복확인하여 숫자 추가
  - List 오름차순 정렬
  - List Size 확인

4. 로또번호 x개 출력

5. 지난 주 당첨번호 입력
예외) 빈칸, 번호가 6개가 아닐 경우, 숫자가 아닐경우, 45이하가 아닐경우

6. 보너스볼 입력
예외) 빈칸, 번호가 1개가 아닐 경우, 숫자가 아닐경우, 45이하가 아닐 경우

7. Comma로 5번에 입력한 번호 분리

8. 당첨확인

9. 수익률 계산

10. 결과출력
